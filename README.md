 ANLP Final Lab — Sentiment Analysis, RAG & Text Generation

This project is the **Final Lab for the Applied Natural Language Processing (ANLP)** course.  
It showcases three main NLP tasks using **Hugging Face Transformers**, **Sentence Transformers**, and **PyTorch**.

---

## **Contents**

1. **Sentiment Analysis** — Classifies the sentiment of text reviews (Positive, Negative, Neutral).  
2. **Retrieval-Augmented Generation (RAG)** — Retrieves the most relevant documents based on semantic similarity.  
3. **Text Generation** — Generates coherent sentences using the GPT-2 model.
